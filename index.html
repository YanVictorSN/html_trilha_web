Canvas

É um elemento HTML que cria uma janela podendo ser usado para manipular objetos usando JavaScript.
Com ele é possível fazer:

  • Desenhos
  • Animações
  • Combinações de Fotos
  • Escritas

Para usarmos Canvas será preciso:
1 - Criar uma tag Script
2 - Criar variável
3 - Chamar o documento a ser desenhado pelo ID (document.getElementByID();
4 - Vincular a tag canvas por seu ID.
